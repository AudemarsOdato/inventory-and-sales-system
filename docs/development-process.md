# Development process
## Step 1: Specifications and Requirements
- What is the project? Description
- What does it do? Functionalities and Specifications
- Why to do this project and for who?
## Step 2: Research and Design
- Designing the UI
- Logic design
- Algorithm design
- System flow
- Research how to implement them
## Step 3: Implementation
- Coding / Writing
- Debugging
- Refactoring code and structure
- Testing
